# Proyecto Final - JavaScript

## Descripción

Este proyecto desarrolla los ejercicios vistos durante las cinco lecciones del módulo 3 de JavaScript, aplicando los conceptos fundamentales del lenguaje mediante ejemplos prácticos.

## Desarrollo de Proyecto Módulo 3

- Lección 1: Introduccion al Lenguaje de Javascript
- Lección 2: CVariables, expresiones y sentencias condicionales.
- Lección 3: Arreglos y ciclos (for y while).
- Lección 4: Funciones en JavaScript.
- Lección 5: Conceptos básicos de objetos de JavaScript

## Tecnologías utilizadas

- HTML5
- JavaScript

## Cómo ejecutar el proyecto

1. Abrir el archivo `index.html` en un navegador web.
2. Presionar `F12`.
3. Abrir la pestaña **Consola** para visualizar los resultados de los ejercicios.
